# agentic-tools

Skills, MCP servers, and home-manager modules for AI coding CLIs.

## What This Provides

- **Stacked workflow skills** — 6 skills for stacked commit workflows
  (Claude Code, Kiro, Copilot)
- **MCP server packages** — 12 servers packaged as Nix derivations with
  typed settings and credential handling
- **Home-manager modules** — declarative config for Copilot CLI, Kiro CLI,
  MCP servers, and stacked workflows
- **DevShell modules** — per-project AI config without home-manager
- **Git tool overlays** — git-absorb, git-branchless, git-revise

Skills work without Nix. Everything else requires Nix.
